My Foodie

My Foody is the best application for those of you who like to cook. You can search and save recipes also filter by meal and diet type. Write in Kotlin, use technology Material Design, Retrofit, Room Database, Dagger Hilt, Coroutine, Flow, Data Store, Glide, and View Pager also implement Bottom Navigation, Action Mode, Bottom Sheet, Motion Layout, Light and Dark Theme and Offline Caching. Link Github: My Foodie.

Write in Kotlin, use technology Room Database, Dagger Hilt, Material Design, Coroutine, Flow, and Navigation also implement Bottom Navigation, Action Mode, and Bottom Sheet. Link github: Todos App.

Find latest earthquake in the world. Using USGS API to fetch the data. Write in Kotlin, use technology Retrofit, Paging 3, Jetpack Compose, Dagger Hilt, Material Design, Coroutine, Flow, and Navigation also implement Google Map API for show the epicentrum. Link Github: USGS Earthquake.
